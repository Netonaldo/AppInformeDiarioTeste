# Contribuindo

Obrigado por considerar contribuir com o **AppInformediario**!

## Como contribuir
1. Faça um fork do repositório.
2. Crie uma branch: `git checkout -b minha-mudanca`.
3. Faça as alterações e rode os testes:
   - `npm run smoke`
4. Faça commit das mudanças: `git commit -m "Minha mudança"`.
5. Envie o push: `git push origin minha-mudanca`.
6. Abra um Pull Request.

## Padrões
- Mantenha alterações pequenas e focadas.
- Descreva claramente o que mudou e por quê.
- Verifique se o app continua funcionando localmente.

## Relatar problemas
Abra uma *Issue* descrevendo o problema, passos para reproduzir e prints se possível.
